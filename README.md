# Social Media Platform

A mini social media web application developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

* User Registration & Login
* Create Posts
* Like Posts
* Comment on Posts
* Follow / Unfollow Users
* Edit Posts
* Delete Posts
* User Profile Page
* Dark Mode
* Instagram-style Navbar
* Responsive UI Design

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## How to Run the Project

### 1. Clone the Repository

```bash
git clone <your-github-repo-link>
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Backend Server

```bash
npx nodemon backend/server.js
```

### 4. Run Frontend

Open `index.html` using Live Server in VS Code.
